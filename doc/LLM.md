# LLM Interactions

## Summary

ChatGPT was used as a development assistant during the implementation of the TopWords project.

## How It Was Used

The LLM provided help with:

- Clarifying project architecture (separating `core` and `app`)
- Designing the sliding window logic and observer pattern
- Setting up command-line parsing using `mainargs`
- Debugging SBT configuration and runtime issues
- Suggesting unit test structure and identifying edge cases

## Verification Process

All suggestions generated by the LLM were manually reviewed, adapted where necessary, compiled, and tested locally.  
Final implementation decisions and validation were performed independently.

## Limitations

The LLM did not have direct access to the full project state and occasionally required corrections (e.g., package names or SBT commands).  
No code was accepted without manual verification.

## Conclusion

The LLM was used strictly as a support tool for guidance and debugging.  
All final code, integration, and testing were completed manually.
